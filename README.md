# React + TypeScript + Vite

Открыть терминал в VS Code, можно в шапке программы "Терминал => Создать терминал", так же можно с помощью HotKeys "ctrl + shift + `" последний знак находится на одной клавише с буквой "Ё" и знаком "~" в основном

Установка зависимостей "npm i" или "npm install"

Запуск программы "npm run dev", запуск на локальном хосте "http://localhost:5173/"

Запуск сборщика Vite "npm run build" в корне рабочей папки создастся папка "dist"

Запуск сборки проекта "npm run prewiew"

Эти все команды можно найти в файле "package.json" в "scripts", запустить наведя курсор на одну из них "dev", "build" появится окошко над ними, нажмите кнопку "Запуск сценария"

Вот так выглядит 
"scripts": {
    "dev": "vite",
    "build": "tsc -b && vite build",
    "lint": "eslint .",
    "preview": "vite preview"
  },
